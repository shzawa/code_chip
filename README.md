# Code Chip

JavaScript による文字列共有ジェネレータ

## Description

[Deflate](https://ja.wikipedia.org/wiki/Deflate)アルゴリズムを用いて、URL の query 部を経由してコードをシェアできる URL ジェネレータ  
markdown 対応したいっすね
